# 📚 AI Study Planner

An intelligent web-based study planner that helps students organize their study schedule by automatically generating a personalized study timetable from uploaded learning materials or manually entered syllabus content. The application includes smart schedule generation, built-in study timers, task tracking, and automatic progress saving to improve productivity and exam preparation.

---

## 🚀 Features

- 🔐 Simple Login Interface
- 📄 Upload TXT, PDF, and DOCX files
- ✍️ Manual syllabus or study material input
- 🤖 Automatic topic extraction
- 📅 Smart study timetable generation
- ⏱️ Manual and automatic study session timer
- 🕒 Built-in analog and digital countdown timer
- ✅ Task completion tracking
- 💾 Automatic data saving using Local Storage
- 📱 Responsive and user-friendly interface
- 🎨 Modern Glassmorphism UI design

## ⚙️ How It Works

1. Login to the application.
2. Enter the subject name.
3. Upload a TXT, PDF, or DOCX file, or paste the syllabus manually.
4. Enter the available study days and study hours.
5. Choose either Manual or Automatic timer mode.
6. Click **Generate Your Table**.
7. Start each study session using the built-in timer.
8. Mark completed topics to track your progress.
